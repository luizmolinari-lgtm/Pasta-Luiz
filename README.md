# 🌾 Projeto Agricultura Sustentável Interativa

Este é um projeto interativo desenvolvido para ajudar as pessoas a entenderem melhor as práticas de agricultura sustentável através de um quiz dinâmico.

## 🚀 Tecnologias Utilizadas
* HTML5
* CSS3
* JavaScript

## 📁 Estrutura do Projeto
* `index.html`: Estrutura da página web.
* `style.css`: Estilização e design responsivo.
* `script.js`: Lógica do quiz interativo.

## 🧑‍💻 Como Executar
1. Faça o clone deste repositório.
2. Abra o arquivo `index.html` no seu navegador.
